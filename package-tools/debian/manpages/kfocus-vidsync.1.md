# NAME
kfocus-vidsync - GUI video sync profile manager

# SYNOPSIS
**kfocus-vidsync**

# DESCRIPTION
**kfocus-vidsync** is a GUI utility using KDialog for managing NVidia video sync profiles on Kubuntu Focus laptops.
