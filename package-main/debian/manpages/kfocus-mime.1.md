# NAME
kfocus-mime - mimetype handler for kfocus.org website

# SYNOPSIS
**kfocus-mime**

# DESCRIPTION
**kfocus-mime** handles downloadable files to launch applications from the
kfocus.org website. **kfocus-mime** is not meant to be run on its own or from
the command line.

# OPTIONS
none
